Match

class Match(source)

A slice of input satisfying a matcher.

Substrings are evaluated lazily upon conversion to a SyntaxTreeNode of the same matcher.

Parameters

begin

the offset of the full sequence where the matched substring begins

endExclusive

one past the last index containing a character in the matched substring

choice

if matcher requires choosing between multiple rules, represents the index of the option chosen. Otherwise, is 0

depth

the depth of the matcher, if nested. If the matcher is not nested, the value of this property is 0
