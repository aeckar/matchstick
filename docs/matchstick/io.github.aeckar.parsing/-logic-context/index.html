LogicContext

class LogicContext : RuleContext, CharSequence(source)

Configures a Matcher that is evaluated each time it is invoked, whose behavior is described by a user-defined function.

Matches captured by an invocation of yield or successive invocations of include are considered explicit.

As a CharSequence, represents the remaining characters in the input.

It is the user's responsibility to ensure that operations on instances of this class are pure. This ensures correct caching of matched substrings.
